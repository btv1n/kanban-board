# Канбан-доска
Веб-приложение для управления задачами в формате канбан-доски с авторизацией по имени, drag-and-drop интерфейсом и хранением данных в файловой системе.

## Функциональный возможности приложения
- вход по имени пользователя (без пароля);
- создание и управление несколькими досками;
- создание, редактирование, и удаление колонок и задач;
- перетаскивание колонок и задач (drag-and-drop);
- загрузка изображений в колонки и перемещение изображений между колонками;
- автоматическое сохранение состояния доски на сервере.

## Быстрый запуск приложения на Windows
1. Установите [Node.js](https://nodejs.org/) на компьютер
2. В корне проекта запустите `install.bat` для установки необходимых компонентов
3. Запустите `start.bat`, чтобы начать работу с проектом
4. В случае если браузер не открыл сервер автоматически, перейдите по адресу `http://localhost:5173`

## Запуск проекта через npm
```bash
npm install
npm run dev
```
## Стек технологии
- **Frontend:** React, Vite
- **Backend:** Node.js, Express
- **Хранение данных:** JSON-файлы в `server/data` (без отдельной СУБД)

## Структура проекта
```text
kanban-board-main/
  src/                 # frontend (React)
  server/              # backend (Express + файловое хранилище)
  install.bat          # установка зависимостей для Windows
  start.bat            # запуск проекта для Windows
```
